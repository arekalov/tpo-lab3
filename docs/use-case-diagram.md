## Акторы

| Актор                        | Описание                                                        |
|------------------------------|-----------------------------------------------------------------|
| Гость                        | Незарегистрированный или не вошедший в аккаунт посетитель сайта |
| Авторизованный пользователь  | Пользователь, выполнивший вход в свой аккаунт                  |
