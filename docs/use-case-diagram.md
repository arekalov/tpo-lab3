```mermaid
flowchart LR
    Guest(["Гость"])
    AuthUser(["Авторизованный\nпользователь"])

    subgraph SO ["StackOverflow"]
        UC01["UC-01: Просмотр главной страницы"]
        UC02["UC-02: Поиск вопроса по ключевым словам"]
        UC03["UC-03: Просмотр карточки вопроса"]
        UC04["UC-04: Регистрация/вход в аккаунт"]
        UC05["UC-05: Создание нового вопроса"]
        UC06["UC-06: Добавление ответа"]
        UC07["UC-07: Голосование за вопрос/ответ"]

        UC09["UC-09: Редактирование вопроса/ответа"]
        UC10["UC-10: Просмотр тегов и переход по тегу"]

    Guest --> UC01
    Guest --> UC02
    Guest --> UC03
    Guest --> UC04

    AuthUser --> UC01
    AuthUser --> UC02

    QuestionAuthor --> UC08
```

## Акторы

| Актор                        | Описание                                                         |
|------------------------------|------------------------------------------------------------------|
| Гость                        | Незарегистрированный или не вошедший в аккаунт посетитель сайта  |
| Авторизованный пользователь  | Пользователь, выполнивший вход в свой аккаунт                   |
| Автор вопроса                | Авторизованный пользователь, являющийся владельцем вопроса       |
