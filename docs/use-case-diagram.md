# Use Case Диаграмма — StackOverflow

```plantuml
@startuml
left to right direction

actor "Гость" as Guest
actor "Авторизованный\nпользователь" as AuthUser

AuthUser --|> Guest

usecase "UC-01: Просмотр главной страницы" as UC01
usecase "UC-02: Поиск вопроса по ключевым словам" as UC02
usecase "UC-03: Просмотр карточки вопроса" as UC03
usecase "UC-04: Регистрация/вход в аккаунт" as UC04
usecase "UC-08: Просмотр тегов и переход по тегу" as UC08

usecase "UC-05: Создание нового вопроса" as UC05
usecase "UC-06: Добавление ответа" as UC06
usecase "UC-07: Голосование за вопрос/ответ" as UC07
usecase "UC-09: Добавление вопроса в избранное" as UC09
usecase "UC-10: Просмотр сохранённых вопросов" as UC10

Guest --> UC01
Guest --> UC02
Guest --> UC03
Guest --> UC04
Guest --> UC08

AuthUser --> UC05
AuthUser --> UC06
AuthUser --> UC07
AuthUser --> UC09
AuthUser --> UC10

' <<include>>: UC-05..09 требуют авторизации
UC05 ..> UC04 : <<include>>
UC06 ..> UC04 : <<include>>
UC07 ..> UC04 : <<include>>
UC09 ..> UC04 : <<include>>
UC10 ..> UC04 : <<include>>

' <<extend>>: действия расширяют просмотр карточки вопроса
UC06 ..> UC03 : <<extend>>
UC07 ..> UC03 : <<extend>>
UC09 ..> UC03 : <<extend>>
@enduml
```

## Акторы

| Актор                        | Описание                                                        |
|------------------------------|-----------------------------------------------------------------|
| Гость                        | Незарегистрированный или не вошедший в аккаунт посетитель сайта |
| Авторизованный пользователь  | Пользователь, выполнивший вход в свой аккаунт                  |

## Прецеденты

| ID     | Название                           | Доступен гостю | Требует авторизации |
|--------|------------------------------------|:--------------:|:-------------------:|
| UC-01  | Просмотр главной страницы          | ✓              |                     |
| UC-02  | Поиск вопроса по ключевым словам   | ✓              |                     |
| UC-03  | Просмотр карточки вопроса          | ✓              |                     |
| UC-04  | Регистрация/вход в аккаунт         | ✓              |                     |
| UC-05  | Создание нового вопроса            |                | ✓ `<<include>>` UC-04 |
| UC-06  | Добавление ответа                  |                | ✓ `<<include>>` UC-04, `<<extend>>` UC-03 |
| UC-07  | Голосование за вопрос/ответ        |                | ✓ `<<include>>` UC-04, `<<extend>>` UC-03 |
| UC-08  | Просмотр тегов и переход по тегу   | ✓              |                     |
| UC-09  | Добавление вопроса в избранное     |                | ✓ `<<include>>` UC-04, `<<extend>>` UC-03 |
| UC-10  | Просмотр сохранённых вопросов      |                | ✓ `<<include>>` UC-04 |
